A Python recreation of the classic Snake game, inspired by Google's Snake design. Built with Pygame, featuring smooth grid-based movement, score tracking, and sound effects. The project also includes an autonomous AI mode that uses pathfinding (BFS) with a survival fallback strategy to play the game on its own — reaching for the apple efficiently while avoiding dead ends.
